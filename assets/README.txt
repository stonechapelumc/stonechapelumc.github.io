Put image files here.

Examples referenced by the site (replace the placeholder blocks in the
HTML with <img> tags pointing at these):

  assets/strawbridge-portrait.jpg   - portrait on the History page
  assets/chapel-exterior.jpg        - building photo on the History page
  assets/cemetery.jpg               - cemetery/parish hall on the History page
  assets/photos/*.jpg               - images for the Historic Photos page
  assets/<firstname>.jpg            - optional headshots on the Leadership page

Use JPG or WebP. Aim for ~1600px on the long edge so pages stay fast.
